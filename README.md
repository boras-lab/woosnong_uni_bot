# Woosong WhatsApp Bot

WhatsApp автоответчик для Woosong University Kazakhstan.

## Установка

### 1. Установите Node.js

Скачайте и установите [Node.js](https://nodejs.org/) (рекомендуется версия 18+).

### 2. Клонируйте репозиторий

```bash
git clone <url_репозитория>
cd woosong-whatsapp-bot
```

### 3. Установите зависимости

```bash
npm install
```

## Запуск

```bash
npm start
```

При первом запуске появится QR-код в консоли. Отсканируйте его приложением WhatsApp на телефоне (WhatsApp → Связанные устройства → Связать устройство).

## Связанные файлы

- `index.js` — точка входа в приложение
- `src/` — исходный код бота
